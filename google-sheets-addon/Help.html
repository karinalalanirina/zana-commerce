Wasnap WhatsApp Shop · Help

Turn this sheet into a live WhatsApp storefront in 3 steps.

1. Set your API key

Open your Wasnap account, find the Sheets API Key section, click Generate. Copy the wsn_live_… token.

Back in Sheets: Extensions → Wasnap WhatsApp Shop → Settings · API key. Paste and save.

2. Lay out your products

Row 1 must contain these column headers (in any order — matched by name, not position):

Product NameCategoryDescriptionImage URLPriceSKUStockActive

One product per row. Image URL must be publicly reachable — Google-Drive share links won't work. Active accepts Y or N. Leave Stock blank for unlimited.

Don't know what data to enter? Click Create shop → Start from demo — we'll populate the sheet with 3 example rows so you can see the shape.

3. Publish

Extensions → Wasnap WhatsApp Shop → Create shop. Review the product list, click Next, enter shop name + currency + WhatsApp number, click Publish. Your shop is live in under 2 seconds — copy the link from the success screen.

To make changes later, edit the sheet and run Update shop from the same menu.

Troubleshooting

  • Menu isn't showing — refresh the spreadsheet tab (Ctrl+R). Newly installed add-ons need a page reload.
  • "No valid key" — your token was revoked or rotated on Wasnap. Open Settings · API key and paste a fresh one.
  • Images show as 📦 on the storefront — the URL isn't publicly reachable. Try https://picsum.photos/seed/xxx/400 for a quick test image.
  • Publish failed: HTTP 401 — your Wasnap account key was revoked. Generate a new one and re-paste.
  • Sync changes my prices on Wasnap — that's by design: the sheet is the source of truth. Edit on Wasnap if you want sheet→Wasnap to stay one-way, then don't re-sync.
